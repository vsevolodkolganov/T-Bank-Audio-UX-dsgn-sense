# Balance Without Words

## Concept

Speaking financial numbers in public spaces can compromise privacy.

Instead of vocalizing the balance, the system converts the amount into **sound texture density**.

This allows the user to understand their financial state **without hearing exact numbers**.

---

# Sound Balance Model

Balance is mapped to audio richness.

| Balance Level | Sound Texture |
|---------------|--------------|
Low | light airy tone |
Medium | layered harmonic pad |
High | dense harmonic texture |

---

# Example Interaction

User asks:

"Какой у меня баланс?"

Public environment detected.

System response:

earcon: privacy_mid

Then plays balance texture.

User instantly understands financial state without hearing numbers.

---

# Sonic Parameters

Variables used:

harmonic layers  
stereo width  
low-frequency presence  
reverb tail

Higher balances produce:

- thicker harmonics
- wider stereo field
- stronger resonance

# Optional Mode

Users can enable **precise mode**, where the assistant speaks exact numbers.

Default mode prioritizes privacy.
