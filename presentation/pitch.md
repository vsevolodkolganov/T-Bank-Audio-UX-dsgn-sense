# T-Bank Audio UX
Pitch Presentation

---

# 1. Problem

Most banking apps depend entirely on screens.

But many everyday situations make screen interaction inconvenient:

walking in the city  
driving  
carrying bags  
public transport

Financial actions become slower and less safe.

---

# 2. Insight

People already interact with technology through sound:

voice assistants  
headphones  
smart speakers

Audio interfaces allow interaction **without visual attention**.

This creates an opportunity for a new type of banking experience.

---

# 3. Solution

T-Bot — a voice-first banking interface.

Users can control their finances through:

voice commands  
audio feedback  
context-aware responses

No screen required.

---

# 4. Key Idea

Sound replaces visual UI elements.

Buttons → earcons  
loading animations → pulses  
notifications → tonal signals

The interface becomes **auditory**.

---

# 5. Smart Silence

Unlike traditional assistants, the system does not talk too much.

T-Bot dynamically adapts between four modes:

Silent  
Micro  
Conversational  
Ambient

This keeps interaction natural and unobtrusive.

---

# 6. Privacy

Financial information is sensitive.

The system includes:

voice biometrics  
spoofing protection  
privacy audio mode

Balance can be communicated using **sound textures instead of numbers**.

---

# 7. Example Interaction

User walking with groceries.

Command:

"Переведи 350 рублей Маше"

System response:

🎧 listening tone  
🎧 processing pulse  
🎧 success chord

Voice:
"Готово."

Total interaction time:

≈ 2 seconds.

---

# 8. Benefits

Faster micro-transactions

Safer interaction while moving

Accessibility for visually impaired users

Stronger emotional brand connection

---

# 9. Why It Matters

Interfaces are gradually moving beyond screens.

Audio UX opens a new interaction paradigm for digital banking.

T-Bot explores how financial services might feel in a **Zero-UI future**.

---

# 10. Vision

The best interface is the one that disappears.

T-Bot turns banking into something simple, fast and almost invisible.
